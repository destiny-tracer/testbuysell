# FinCalc – Finance Calculator Web App

A modern, responsive, **TypeScript-based** finance calculator web application designed to run entirely on **GitHub Pages** with **no backend** and **no build tools**.  
Includes multiple calculators, charts, and debt payoff strategies with a clean, professional UI and **dark mode support**.

---

## ✅ Features
- **Modern UI**: Apple/Notion-inspired design, responsive grid layout.
- **Dark Mode**: Toggleable via navbar.
- **Customizable Theme**: CSS variables for easy color changes.
- **SPA Navigation**: Uses URL parameters (`?id=calculatorName`) for loading calculators.
- **Charts**: Integrated with Chart.js (via CDN).
- **Debt Strategies**: Snowball, Avalanche, Hybrid payoff simulations.
- **Modular Architecture**: Easy to add new calculators.
