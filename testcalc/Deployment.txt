Step 1: Install TypeScript
You need TypeScript to compile .ts files into .js files.
Run this in your terminal:
npm install -g typescript


✅ Step 2: Initialize TypeScript Config
Inside your project folder (/fincalc), run:
tsc --init

This creates a tsconfig.json file. Replace its content with:
{
  "compilerOptions": {
    "target": "ES6",
    "module": "ES6",
    "outDir": "./",
    "rootDir": "./",
    "strict": true
  },
  "include": ["./**/*.ts"]
}


✅ Step 3: Compile TypeScript
Run:
tsc

This will generate .js files alongside your .ts files.

✅ Step 4: Update index.html
Make sure all <script> tags in index.html point to the compiled .js files, not .ts.
Example:
./app.js</script>


✅ Step 5: Push Files to GitHub

Create a new GitHub repository.
Upload:

index.html
style.css
All compiled .js files
README.md
LICENSE


Commit and push.


✅ Step 6: Enable GitHub Pages

Go to Settings → Pages in your GitHub repo.
Under Source, select:

Branch: main
Folder: /root


Click Save.

Your app will be live at:
https://<your-username>.github.io/<repo-name>/


✅ Step 7: Test Your App

Open the URL.
Check SPA navigation (?id=calculatorName).
Verify dark mode toggle and calculators.
